# Transaction Dispute Portal

A .NET 10 application for managing transaction disputes.

## Project Structure

- **backend/TransactionDisputePortal.Api** - Backend API project

## Getting Started

### Prerequisites
- .NET 10 SDK
- Visual Studio 2026 or later (or VS Code)

### Build and Run

```bash
dotnet build
dotnet run --project backend/TransactionDisputePortal.Api/TransactionDisputePortal.Api.csproj
```
